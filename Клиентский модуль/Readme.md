# Car Wash Booking App

Мобильное приложение для онлайн-бронирования услуг автомойки с предоплатой, управлением загрузкой постов и масштабируемой архитектурой под SaaS.

Проект разрабатывается поэтапно: от минимального работающего прототипа до полноценной облачной платформы.

---

## Цель проекта
Создать надёжную и масштабируемую систему бронирования автомоек, которая:
- исключает неявки за счёт предоплаты,
- выравнивает загрузку постов,
- повышает средний чек,
- готова к дальнейшему масштабированию (multi-tenant, Kubernetes).

---

## Текущий этап
**MVP-0** — клиентский блок (минимальный функционал).

Фокус этапа:
- отладка логики бронирования,
- проверка связей между интерфейсом и базой данных,
- валидация базовой бизнес-логики.

Без:
- реальной оплаты,
- аутентификации,
- уведомлений,
- админских и аналитических функций.

---

## Основные модули (план)
- Client (клиентское приложение)
- Admin (управление записями)
- Worker (исполнение услуг)
- Owner (аналитика и контроль бизнеса)

На текущем этапе реализуется только **Client**.

---

## Архитектурный подход
- Модульная архитектура
- Чёткое разделение ролей
- Готовность к контейнеризации и масштабированию
- Минимальный MVP → постепенное наращивание функционала
